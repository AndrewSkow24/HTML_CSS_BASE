Наследование CSS-свойств

Пусть у нас есть абзац, а в нём какой-то текст в теге i

This is some text in paragraph.

Если зададим абзацу красный цвет текста, то и то, что внутри тега i также покрасится в красный цвет.

Всё дело в том, что свойство color наследуется. Это значит, что если у родительского тега задан какой-то цвет, то такой же цвет будет и у потомков. Наследуются не все свойства CSS, но многие (из тех, что вы уже знаете - все наследуются).

При желании, однако, можно переопределить свойства родителя задав селектор потомка. Например давайте абзацу зададим красный цвет, а курсиву - голубой.

This is some text in paragraph.

Порядок селекторов в CSS коде в таком случае не имеет значения. Если переставить местами селектор i и селектор p, всё будет работать точно также.

Задача 1

Изучив код, скажите какой цвет будет иметь текст в теге b:

This is some text in paragraph.

p { color: blue; }

Цвет содержимого тега b тоже будет синий, так как он наследуется от родителя. А отдельный цвет для b не задан.

Задача 2

Изучив код, скажите какой цвет будет иметь текст в теге b:

This is some text in paragraph.

p { color: blue; } b { color: red; }

Тег b будем иметь красный цвет, т.к. для него задан отдельный селектор отличный от родительского.

Задача 3

Изучив код, скажите какой цвет будет иметь текст в теге b:

This is some text in paragraph.

b { color: red; } p { color: blue; }

От перемены мест тегов результат не изменится

Задача 4

Изучив код, скажите какой цвет будет иметь текст в теге li:

  • text
  • text
  • text
ul { color: blue; }

Свойство color будет унаследовано от родительского тега ul, поэтому цвет текста содержимого li будет синим.

Задача 5

Изучив код, скажите какой цвет будет иметь текст в теге li:

  • text
  • text
  • text
ul { color: blue; } li { color: red; }

В данном случае цвет будет красным.

Задача 6

Изучив код, скажите какой цвет будет иметь текст в теге i:

  • text italic text
  • text italic text
  • text italic text
ul { color: blue; } li { color: red; } i { color: green; }

Ссылки будут зелёными, так как для них задано свойство в отдельном селекторе.

Задача 7

Изучив код, скажите какой размер будет иметь текст в теге i:

  • text italic text
  • text italic text
  • text italic text
i { font-size: 30px; } ul { font-size: 20px; } li { color: red; font-weight: bold; }

Размер тега будет 30px, остальные свойства будут унаследованы от родителей.
