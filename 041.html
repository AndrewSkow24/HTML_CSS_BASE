Свойство-сокращение font в CSS

В предыдущем уроке мы разобрали много свойств для текста. Зачастую достаточно накладно отдельно прописывать каждое из этих свойств, поэтому в CSS для большего удобства существует специальное свойство сокращение font. Данное свойство позволяет одновременно задавать:

  1. размер шрифта
  2. семейство
  3. жирность
  4. курсив
  5. межстрочный интервал

Описываемое свойство имеет следующий синтаксис:

  1. курсив
  2. жирность
  3. размер
  4. / межстрочный интервал
  5. семейство

При этом порядок свойств имеет значение, и обязательными являются размер шрифта и семейство. Обратите внимание на то, что другие свойства, кроме описанных выше в синтаксисе, не входят в данное Свойство-сокращение.
